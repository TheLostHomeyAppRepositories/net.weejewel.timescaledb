Are you hungry for more data? Log your device's capability values to your self-hosted TimescaleDB!

With a filled database, it should be easy to connect with Grafana or other visualization tools to run powerful queries.

USAGE
After installing the app, visit the app's settings.

Within the Homey app for iOS/Android, navigate to More (⋯) → Settings → TimescaleDB.
Within the Homey Web App, navigate to Settings (⚙︎) → TimescaleDB → Configure.

Then enter your TimescaleDB URI to connect. You'll see every Number- and Boolean-type device capability change automatically appear in your database.
